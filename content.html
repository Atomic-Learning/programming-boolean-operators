Combining Boolean Values

Boolean operators allow us to combine multiple boolean values or expressions to create more complex conditions. The three primary boolean operators supported by most programming languages are not, and, and or.

The NOT Operator

The not operator reverses the boolean value—it returns true if the input is false, and false if the input is true.

a = true
not a → false

b = false
not b → true

The AND Operator

The and operator returns true only if both operands are true. If either (or both) operand is false, the result is false.

true and true   → true
true and false  → false
false and true  → false
false and false → false

The OR Operator

The or operator returns true if at least one operand is true. It returns false only if both operands are false.

true or true    → true
true or false   → true
false or true   → true
false or false  → false
